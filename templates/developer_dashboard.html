Quick Start Implementation

Our asynchronous API allows you to submit scan jobs and retrieve results when ready. Follow this pattern for reliable integration.

  • Submit a Scan Job

    POST a URL to the /api/extern/scan endpoint.

    import requests

    url = "https://{{ request.host }}/api/extern/scan"
    headers = {
      "X-API-KEY": "{{ api_key.key if api_key else 'YOUR_KEY' }}"
    }
    payload = { "url": "https://shop.tiktok.com/..." }

    resp = requests.post(url, json=payload, headers=headers)
    job_data = resp.json()
    job_id = job_data.get('job_id')
    print(f"Job Started: {job_id}")
  • Poll for Status

    Check the status endpoint until status is 'completed'.

    import time

    status_url = f"https://{{ request.host }}/api/extern/jobs/{job_id}"

    while True:
      r = requests.get(status_url, headers=headers)
      data = r.json()
      if data['status'] == 'completed':
        print(data['result'])
        break
      elif data['status'] == 'failed':
        print("Error processsing")
        break
      time.sleep(2)
