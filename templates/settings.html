EchoTik API Configuration

These settings are configured via environment variables on your hosting platform (Render.com).

To update your API credentials or scan settings, go to your Render dashboard and update the environment variables.

Environment variable: ECHOTIK_USERNAME

Environment variable: ECHOTIK_PASSWORD

Scan Settings

Configure these via environment variables on Render:

SCAN_REGION - Default: US

Options: US, GB, ID, TH, PH, MY, VN, SG

PAGES_TO_SCAN - Default: 20 (200 products)

MIN_SALES - Default: 10

MAX_INFLUENCERS - Default: 500

MIN_COMMISSION - Default: 0.10 (10%)

Daily Scan Schedule

The daily scan runs automatically every day via Render's Cron Job feature.

Default schedule: 9:00 AM UTC (configured in render.yaml)

To change the schedule, update the schedule field in render.yaml and redeploy.
