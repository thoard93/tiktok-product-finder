How to get your cookie:

  1. Log in to EchoTik
  2. Open DevTools (F12) -> Application tab -> Cookies
  3. Find echotik.live -> Copy the entire Cookie header value.
  4. Pro Tip: Copy the whole string from your browser's network request for the most reliable results.

How to get your token:

  1. Log in to DailyVirals
  2. Open DevTools (F12) -> Network tab
  3. Refresh page & find any request to backend.thedailyvirals.com
  4. Find Authorization header -> Copy the long string after 'Bearer '
